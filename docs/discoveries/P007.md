## Sai lệch so với phiếu

### SD-1 [mechanical] — `predicates::str::contains(..).or(..)` requires trait import

**Phiếu spec:** `assert.stderr(contains("format").or(contains("ISO")))` (phiếu line 959)  
**Thực tế:** Trait `PredicateBooleanExt` phải được import tường minh để dùng `.or()` trên `ContainsPredicate`. Clippy báo `E0599` lúc compile.  
**Xử lý:** Thêm `use predicates::prelude::PredicateBooleanExt;` vào integration test. Tầng 2 self-decide — không thay đổi behavior, chỉ thêm import.

### SD-2 [mechanical] — clippy `io_other_error` lint

**Phiếu spec:** `std::io::Error::new(std::io::ErrorKind::Other, source.to_string())` cho serde serialize error mapping  
**Thực tế:** clippy `-D warnings` bắt `clippy::io_other_error` — yêu cầu dùng `std::io::Error::other(...)` API mới.  
**Xử lý:** Thay bằng `std::io::Error::other(source.to_string())`. Tầng 2 self-decide — same semantics, cleaner API.

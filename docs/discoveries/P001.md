## Sai lệch giữa phiếu và thực tế

### 1. rustfmt line-width wrap (mechanical, non-blocking)

**Phiếu viết:** `main.rs` match arm cho `StateBackfill`, `ScanAndAppend`, `Init` dùng single-line form.

**Thực tế:** rustfmt (edition 2024) wrap các match arm có struct destructuring nhiều field xuống multi-line khi vượt 100 ký tự. Ba arm phải refactor:
- `Commands::StateBackfill { state, inbox, dry_run } => ...` → multi-line
- `Commands::ScanAndAppend { report, inbox, state } => ...` → multi-line
- `Commands::Init { inbox_path, state_path } => ...` → multi-line

**Hành động:** sửa trực tiếp trong EXECUTE, không cần escalate. Behavior không đổi, chỉ formatting.

## Sai lệch so với phiếu

1. **Constraint #10 wrong file (mechanical, self-resolved):** Phiếu cited `cli/append.rs` for exhaustive match. Actual location: `main.rs` `Commands::Append` arm. Fixed in Task 3 scope.

2. **Clippy `if_same_then_else` on `StateBackfill` dispatch:** Phiếu's suggested code had two separate `else if` branches both returning `1` (InboxError and StateReadError). Clippy `-D warnings` rejects this pattern. Fixed by collapsing to `|| ` operator: `InboxError.is_some() || StateReadError.is_some() → 1`. Tầng 2 self-decide.

3. **Test count: 59 (not 57+):** Phiếu said "≥57". Actual: 39 unit + 20 integration = 59. The 6 inbox unit tests (vs. ≥4 specified) and the extra stop-at-next-heading test are all within spec.

4. **`parse_rows` comment block handling — same-line `<!-- -->` vs. phiếu pseudocode:** Phiếu had a slightly inconsistent `in_comment` toggle pseudocode (toggle on, toggle off, then `continue`). Worker implemented cleaner: if `starts_comment && !ends_comment` → set `in_comment = true`; if `starts_comment` (regardless of ends_comment) → skip line. This correctly handles both same-line `<!-- foo -->` and multi-line `<!-- ... -->`. Tầng 2 self-decide.
